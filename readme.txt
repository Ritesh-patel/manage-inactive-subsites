A simple plugin for WordPress multisite to manage inactive subsites.

== Description ==

A simple plugin for WordPress multisite that will allow a super admin to automate what happens to inactive subsites on a public-signup multisite network.

== Installation ==

* Download the plugin zip and upload the zip from Add Plugins screen.
* Alternatively, you can download the plugin, unzip it and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the `Plugins` section.
